高性能即时通信与聊天平台

Spring Boot / Netty / Redis / RabbitMQ / PostgreSQL

基于 Spring Boot + Netty 实现即时通信系统，支持长连接管理、可靠消息投递、离线消息补发和在线用户管理

设计用户、频道、消息、会话与投递日志等核心数据模型，使用 PostgreSQL 持久化聊天记录并支持历史消息查询

使用 Redis 缓存用户在线状态、会话路由、离线消息和待 ACK 状态，减少数据库访问并提升系统响应速度

引入 RabbitMQ 作为第二阶段优化组件，用于消息异步落库、削峰与跨节点事件广播，提高系统吞吐能力并降低主链路延迟

实现 JWT 鉴权、自定义消息协议、应用层双向心跳、ACK 确认机制与统一异常处理，提升系统可靠性与可维护性
