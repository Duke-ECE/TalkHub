# TalkHub Docs

TalkHub is a high-performance instant messaging and chat platform. This site documents setup, architecture, and feature updates.

If you want the shortest path to a working local demo, start with [Mock Quickstart](/en/guide/mock-quickstart).

## Start Here

- [Mock Quickstart](/en/guide/mock-quickstart)
- [Getting Started](/en/guide/getting-started)
- [Architecture](/en/guide/architecture)
- [Login Update](/en/reference/login-update)

## Tech Stack

- Frontend: React + Vite + Tailwind CSS
- Backend: Spring Boot + Netty
- Middleware: Redis + RabbitMQ
- Database: PostgreSQL
